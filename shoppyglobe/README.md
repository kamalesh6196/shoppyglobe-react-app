# ShoppyGlobe Ecommerce App

React ecommerce application built using:

React
Redux Toolkit
React Router
Vite

Features:

Product listing
Search products
Product details page
Add to cart
Update quantity
Checkout
Place order
Lazy loading
Routing
Responsive UI

Steps to run:

npm install
npm run dev
